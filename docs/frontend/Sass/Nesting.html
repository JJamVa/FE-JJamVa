Nesting

중첩(Nesting)​

  • HTML의 태그의 계층 방식과 같이 CSS를 중첩 작성
  • CSS코드가 구조화 되고 가독성 높아진다.
HTML코드
<section>
  <ul>
    <li>안녕하세요.</li>
  </ul>
</section>
Sass코드
section {
  background-color: skyblue;
  ul {
    list-style: none;
    li {
      color: slateblue;
    }
  }
}
Sass 전처리기 후 CSS 코드
section {
  background-color: skyblue;
}
section ul {
  list-style: none;
}
section ul li {
  color: slateblue;
} /*# sourceMappingURL=Practice.css.map */
info

중첩(Nesting)을 사용하는 이유​

중첩(Nesting)을 사용하면 최상위 태그를 한번만 선언하여 자식 태그에게 스타일을 적용할 수 있어 코드 반복 줄임. 단, 계층이 너무 깊게 코드를 작성할 경우 코드 분석이 어려울 수 있다.

속성 중첩(Nesting)​

  • 선택자의 중첩 뿐만이 아닌 스타일 속성에게도 적용이 가능
  • 스타일의 속성을 사용 할 때는 : 꼭 사용 해야된다.
Sass코드
section {
  background: {
    color: red;
    position: center center;
    size: 14px 14px;
  }
}
Sass 전처리기 후 CSS 코드
section {
  background-color: red;
  background-position: center center;
  background-size: 14px 14px;
} /*# sourceMappingURL=Practice.css.map */

ampersand(&)​

  • &는 상위 부모선택자를 가르킴

가상요소, 가상클래스, class, id 등 참조​

예시
Sass코드
ul {
  li {
    &:hover {
      background: skyblue;
      cursor: pointer;
    }
    &:first-child {
      border-bottom: 2px solid red;
    }
  }
}
Sass 컴파일 후 CSS코드
ul li:hover {
  background: skyblue;
  cursor: pointer;
}
ul li:first-child {
  border-bottom: 2px solid red;
} /*# sourceMappingURL=Practice.css.map */

공통 class명을 가진 선택자들을 중첩​

예시
HTML코드
<ul>
  <li class="li-yellow">1</li>
  <li class="li-green">2</li>
  <li class="li-red">3</li>
</ul>
Sass코드
ul {
  .li {
    &-yellow {
      background: yellow;
    }
    &-red {
      background: red;
    }
    &-green {
      background: green;
    }
  }
}
Sass 컴파일 후 CSS코드
ul .li-yellow {
  background: yellow;
}
ul .li-red {
  background: red;
}
ul .li-green {
  background: green;
} /*# sourceMappingURL=Practice.css.map */

중첩이 깊어지면 최상위 부모 선택자로부터 참조​

예시
HTML코드
<section class="sec">
  <h1>안녕하세요.</h1>
  <div class="wrapper">
    <p>반갑습니다.</p>
    <div class="inner">
      <div class="core">Hello</div>
    </div>
  </div>
</section>
Sass코드
.sec {
  height: 300px;
  width: 300px;
  h1 {
    color: #eee;
  }
  .wrapper {
    p {
      color: red;
    }
    .inner {
      & .core {
        background-color: skyblue;
      }
    }
  }
}
Sass 컴파일 후 CSS코드
.sec {
  height: 300px;
  width: 300px;
}
.sec h1 {
  color: #eee;
}
.sec .wrapper p {
  color: red;
}
.sec .wrapper .inner .core {
  background-color: skyblue;
} /*# sourceMappingURL=Practice.css.map */

@at-root​

  • @at-root를 이용하면 중첩에서 벗어난다.
  • 중첩된 선택자에게만 사용가능
예시
HTML코드
<ul class="one two three">
  <li class="one"></li>
  <li class="two"></li>
  <li class="three"></li>
</ul>
Sass코드
.one {
  background-color: black;
  .two {
    @at-root .three {
      background-color: skyblue;
    }
  }
}
Sass 컴파일 후 CSS코드
.one {
  background-color: black;
}
.three {
  background-color: skyblue;
} /*# sourceMappingURL=Practice.css.map */
