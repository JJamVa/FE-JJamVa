SVG 속성​

fill​

  • SVG에서 Stroke영역을 제외한 남은 공간(배경)에 대한 색을 지정

stroke​

  • SVG를 표현하기 위한 선의 색을 표현

stroke-width​

  • SVG를 표현하기 위한 선의 색의 크기(굵기)를 설정

stroke-linecap​

  • SVG의 stroke 양쪽선의 모양을 설정하는 속성

stroke-linejoin​

  • SVG의 stroke가 꺽인선 영역에 대해 모양 설정
note

위의 사진의 선을 SVG의 stroke라고 보았을 때,
빨간 동그라미 부분은 stroke-linecap을 이용하여 모양을 설정
파란 동그라미 부분은 stroke-linejoin을 이용하여 꺽인선 부분의 모양을 설정

stroke-dasharray​

  • stroke의 표현영역과 공백영역을 표현하기 위해 사용하는 속성
  • stroke-dasharray: (표현할 줄의 길이) (공백 간격)
path {
  stroke-dasharray: 10 20;
  /* 10만큼 표현 20만큼 공백 */
}
<svg
  width="320"
  height="2"
  viewBox="0 0 320 2"
  fill="none"
  xmlns="http://www.w3.org/2000/svg"
>
  <path d="M0 1H320" stroke="black" />
</svg>

stroke-dashoffset​

  • stroke의 표현 시작점을 지정하는 속성
  • stroke-dasharray의 시작점을 나타내기 때문에 stroke-dasharray속성이 없다면 동작하지 않는다.
stroke-dashoffset 예시
path {
  stroke-dasharray: 30 30;
  stroke-dashoffset: 0;
}
직선 SVG
<svg
  width="320"
  height="2"
  viewBox="0 0 320 2"
  fill="none"
  xmlns="http://www.w3.org/2000/svg"
>
  <path d="M0 1H320" stroke="black" />
</svg>

stroke-dashoffset: 0​

stroke-dashoffset: 30​

stroke-dashoffset: 50​

위의 화면들을 보면 값이 증가할수록 왼쪽으로 사라지고 오른쪽에 삽입되는 것처럼 보인다.
즉, 시작점 0을 기준으로 stroke-dashoffset의 속성값 만큼 떨어진 지점에서부터 화면 표현한다는 것이다.
stroke-dashoffset의 속성값만큼 잘린 stroke 영역은 기존의 stroke영역 맨뒤에 들어간다.
