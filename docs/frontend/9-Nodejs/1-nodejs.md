### Node.js의 장/단점

- 장점 
1. 싱글쓰레드, 논블로킹 I/O에 따른 빠른속도<br/> 
2. 컴퓨터 자원이 덜 사용<br/>

4. JS 사용하기때문에 FE개발자가 사용 용이<br/> 
5. 생산성이 높음

- 단점
1. 싱글 스레드 기반이라 자원을 많이 먹는 작업오면 과부화
2. CPU한개만 사용(게임 서버로 부적합)
3. 로직이 복잡하면 콜백함수에 갇힘
4. 에러가 발생하면 프로세스가 죽는다.
